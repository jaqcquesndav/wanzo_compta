Dans Auth0, vous pouvez définir des rôles et leurs descriptions via l'interface utilisateur ou l'API. Voici une suggestion pour les rôles et leurs descriptions :

superadmin

Description : A le contrôle total sur l'application, y compris la gestion des utilisateurs, des rôles et des paramètres critiques.
admin

Description : Gère les utilisateurs et les paramètres de l'application, mais avec des permissions limitées par rapport au superadmin.
user

Description : Utilisateur standard avec accès aux fonctionnalités de base de l'application.
